# Security Policy

## Reporting a Vulnerability

If you've found a security issue in this repository or the live site at [portfolio.rashadhussain.com](https://portfolio.rashadhussain.com), please report it privately.

**Contact:** shad@tuta.com

Please include:

- A description of the issue
- Steps to reproduce, or a proof of concept
- The affected URL, endpoint, or file
- Your name or handle if you'd like credit (optional)

I'll acknowledge receipt within 7 days and aim to confirm or dispute the report within 30 days.

## Scope

In scope:

- The application code in this repository
- The live site at portfolio.rashadhussain.com and the underlying Cloud Run service
- The GitHub Actions deploy pipeline

Out of scope:

- Findings that require a compromised GitHub or GCP account to exploit
- Reports based solely on automated scanner output without a working proof of concept
- Denial-of-service attacks
- Social engineering against the maintainer or hosting providers

## Coordinated Disclosure

Please do not publicly disclose the issue until I've had a reasonable chance to address it. I'll credit good-faith reporters in the fix commit message or release notes if requested.
